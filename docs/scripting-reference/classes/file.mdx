# üìÅ File

A File represents an entry to a system file.

import { Structs, BasicType, AuthorityType } from '@site/docs/components/_nanos.mdx';

:::info

This is a **Server** side class. It is not possible to open files from outside the server folder. All path must be relative to the Server‚Äôs executable folder. All files are opened as binary file by default.

:::

## Example


```lua
local configuration_file = File("my_awesome_configuration.json")

local configuration_file_json = JSON.parse(configuration_file:Read(configuration_file:Size()))
```
