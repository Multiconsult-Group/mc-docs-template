# Base Paintable

Base Paintable Class

import { Structs, BasicType, AuthorityType, Enums, Classes } from '@site/docs/components/_nanos.mdx';

A **Paintable** class is a base Class in nanos world which provides customization on the Meshes Materials, exposing common functions to allow you to set custom Material Parameters and loading Textures from Disk.

:::info

This is a base Class. You cannot spawn it directly.

:::

### `SetMaterial`

> Replace a materials of this actor with another Material

```lua
my_entity:SetMaterial(new_material, index)
```

| Type | Parameter | Default | Description |
| :--- | :--- | :--- | :--- |
| <BasicType.String /> | material_path | | The new Material to apply |
| <BasicType.Number /> | index | `-1` | The index to apply - -1 means all indexes |

### <AuthorityType.ClientOnly />`SetMaterialFromWebUI`

> Replace a materials of this actor with a WebUI Material

```lua
my_entity:SetMaterialFromWebUI(webui, index)
```

| Type | Parameter | Default | Description |
| :--- | :--- | :--- | :--- |
| <Classes.WebUI /> | webui | | The WebUI entity to use as Material |
| <BasicType.Number /> | index | `-1` | The index to apply - -1 means all indexes |

### `SetMaterialColorParameter`

> Sets a Color parameter in this actor’s material


```lua
my_entity:SetMaterialColorParameter(parameter_name, color)
```

| Type | Parameter | Description |
| :--- | :--- | :--- |
| <BasicType.String /> | parameter_name | The name of Material parameter |
| <Structs.Color /> | color | The value to set |

### `SetMaterialScalarParameter`

> Sets a Scalar parameter in this actor’s material


```lua
my_entity:SetMaterialScalarParameter(parameter_name, number)
```

| Type | Parameter | Description |
| :--- | :--- | :--- |
| <BasicType.String /> | parameter\_name | The name of Material parameter |
| <BasicType.String /> | scalar | The value to set |

### `SetMaterialTextureParameter`

> Loads a image from computer \(relative to Assets/ folder\) and sets as parameter in this actor’s material


```lua
my_entity:SetMaterialTextureParameter(parameter_name, texture_path)
```

| Type | Parameter | Description |
| :--- | :--- | :--- |
| <BasicType.String /> | parameter\_name | The name of Material parameter |
| [SpecialPath](./scripting-reference/glossary/basic-types.md#specialpath) | texture\_path | The path of the Texture |
