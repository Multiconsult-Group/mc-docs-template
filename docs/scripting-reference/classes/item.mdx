## Usage


```lua
local NewItem = Item(
  Vector(-900, 185, 215),
  Rotator(0, 90, 90),
  "nanos-world::SM_Torch",
  CollisionType.Normal,
  true,
  HandlingMode.SingleHandedMelee,
  ""
)
```
