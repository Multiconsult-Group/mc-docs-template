# Color

A color composed of components (R, G, B, A) with floating point precision.

import { Structs, BasicType } from '@site/docs/components/_nanos.mdx';

:::info

**Tip:** This structure is Open Sourced at [https://github.com/nanos-world/nanos-world-lua-lib](https://github.com/nanos-world/nanos-world-lua-lib). Feel free to push merge requests and suggest changes!

:::

:::tip

**Note:** This structure supports `+`, `-`, `*`, `/`, `==`, and `tostring` operations.

:::

## Usage

```lua
local grey_color = Color(0.5, 0.5, 0.5, 1)
```

## Properties

| **Value** | **Name** | **Description** |
| --------- | -------- | --------------- |
| <BasicType.Number /> | **`R`** | Red color percentage (0-1) |
| <BasicType.Number /> | **`G`** | Green color percentage (0-1) |
| <BasicType.Number /> | **`B`** | Blue color percentage (0-1) |
| <BasicType.Number /> | **`A`** | Alpha transparency percentage (0-1) |
